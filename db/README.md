# DB
All code in here is generated by sqlc from the files and config in the `_sqlc` subfolder

## Installation
https://docs.sqlc.dev/en/latest/overview/install.html

## Making Changes
DON'T MODIFY THE GO FILES DIRECTLY

DO:
1. Modify `schema.sql` to update models
2. Modify `query.sql` to specify queries you want to generate code for
After doing 1 or 2, run `sqlc generate` in the `_sqlc` directory to regenerate the code

## Loading Model Changes into the DB
* BE CAREFUL NOT TO WIPE OUT EXISTING DATA *
psql -h localhost -W -U [username] -d [db_name] -f schema.sql
